CME 193 - Introduction to Scientific Python

Project

The project is meant for you to demonstrate the skills you have learnt by taking this class. You are free in the choice of your project. However, picking the right project can be hard – we'll discuss the types of projects to consider. I'll also update this website as we progress.

The goal of the project should be to have fun. You take this class because you want to learn about Python, and this is your opportunity to use Python in any way you like. The project should take you around 20 hours of work as a rough estimate (as you're brainstorming ideas).
